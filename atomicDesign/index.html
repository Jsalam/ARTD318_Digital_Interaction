GUI System Prototyping with P5.js and JavaScript

This page presents an example of an interactive system built with P5.js developed with the atomic design framework.

Atomic Design is simple conceptual framework developed by Brad Frost that argues for the construction of complex interactive systems from simple GUI building blocks. The framework is a metaphore derived from chemistry and biology. The idea is that molecules are constituted by atoms, sets of molecules form organisms, and collections of organisms result in templates that are used to build interactive systems.

In the construction of this example we follow the principles of Material Design. All its components could be interpreted as atoms or molecules.

Atoms

The fundamental atomic elements of this example are interactive rectangles and circles. These elements react to the mouse proximity and change color whenever the mouse is on top of them.
